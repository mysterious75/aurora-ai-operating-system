# Aurora AI Operating System

Intelligent Business Analysis for Aurora Office Furniture

An AI-powered dashboard that connects to Odoo ERP data, analyzes business metrics, and provides prioritized action items for management.

## Features

- Real-time Dashboard - Sales, orders, inventory at a glance
- AI Analysis - Automated business insights with priority actions
- Inventory Alerts - Stock issues identified automatically
- AI Assistant - Natural language Q&A about your business
- Action Items - Prioritized next steps for the team

## Built For

Aurora Office Furniture (Aust) Pty Ltd
- Canberra, Sydney, Melbourne
- Managing Director: Dean Grace
- Odoo ERP Integration Ready

## Tech Stack

- Frontend: Streamlit
- AI Engine: Anthropic Claude API
- Data Source: Odoo ERP (mock data for demo)
- Deployment: HuggingFace Spaces (free)
